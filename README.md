# Library_managment_system
This project is a Library Management System implemented using Python.
It demonstrates the practical application of Data Structures such as
Linked List, Tree, and Merge Sort.

## Features
- Add new books
- Delete books
- Search books
- Display all books
- Sort books using Merge Sort

## Data Structures Used
- Linked List for storing book records
- Tree for organizing books
- Merge Sort for sorting operations

## Technologies
- Python
- Data Structures & Algorithms

## How to Run
1. Clone the repository
2. Run library.py using Python
3. Follow on-screen menu
